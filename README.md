### Instruction to Run

Run wsl and install this specific Ubuntu distro.
```
wsl --install -d Ubuntu-22.04
```

After install the Ubuntu distro, use visual studio code to reopen the folder in wsl and open a terminal and run the install_cuda.sh using command below:
```
./install_cuda.sh
```
Install CMake Tools Extension in Visual Studio Code and use it to build the project.
